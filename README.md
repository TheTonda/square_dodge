# square_dodge
🕹️ A simple 2D dodging game. Play it live! (🤖 AI-generated)

## 🕹️ How to Play
- **Move:** Use the **Arrow Keys** or **WASD** to navigate your square.
- **Goal:** Avoid the falling red squares for as long as possible.
- **Difficulty:** The game gets faster the longer you survive!


## 🛠️ Built With
- **HTML5 Canvas** for rendering
- **CSS3** for styling
- **Vanilla JavaScript** (No frameworks needed!)
